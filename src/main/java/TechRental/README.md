# Tech Rentals Unlimited

A JavaFX application to manage tech rental inventory for a small business.

## Project Description
Tech Rentals Unlimited allows users to browse available tech items, add new inventory, remove items, and view rental statistics.

## Getting Started Guide

### Prerequisites
- [x] Java 17 or higher
- [x] JavaFX SDK
- [x] MySQL database
- [x] IntelliJ IDEA

### Setup
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Update `TechRental/database/Const.java` with your credentials
4. Run `Launcher.java`
